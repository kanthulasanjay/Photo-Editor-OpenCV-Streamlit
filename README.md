# 🖼️ Photo Editor using OpenCV & Streamlit

## 📌 Project Description

This project is a simple yet powerful Photo Editing Web Application built using OpenCV and Streamlit. Users can upload an image and apply various image processing techniques interactively using sliders and buttons.

---

## 🛠️ Tools & Technologies Used

* Python
* OpenCV
* Streamlit
* NumPy
* Pillow

---

## ✨ Features

* Upload Image
* Resize Image
* Adjust Brightness & Contrast
* Convert to Grayscale
* Apply Blur Effect
* Apply Sharpen Effect
* Warm Filter Effect
* Portrait Background Blur
* Edge Detection (Extra Feature)
* Image Rotation (Extra Feature)
* Download Edited Image

---

## 🔄 Application Flow

Upload Image → Adjust Brightness & Contrast → Apply Filters → View Edited Image → Download Image

---

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/KanthulaSanjay/Photo-Editor-OpenCV-Streamlit.git
```

### 2. Navigate to the project folder

```bash
cd Photo-Editor-OpenCV-Streamlit
```

### 3. Install the required dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit application

```bash
streamlit run app.py
```
