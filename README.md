# PixelForge - Photo Editor

## Description
A Photo Editor built using Streamlit and OpenCV.

## Features
- Upload Image
- Resize
- Brightness
- Contrast
- Grayscale
- Blur
- Warm Filter
- Portrait Blur
- Sharpen
- Download Image

## Technologies
- Python
- Streamlit
- OpenCV
- NumPy
- Pillow

## Run

pip install -r requirements.txt

streamlit run app.py
